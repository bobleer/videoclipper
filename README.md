# videoclipper
The easiest video clipper.  
  
# Install
```sh
curl 'https://raw.githubusercontent.com/bobleer/videoclipper/master/videoclipper.sh' -o /usr/local/bin/videoclipper && chmod +x /usr/local/bin/videoclipper
```

# Usage
```sh
videoclipper
```
